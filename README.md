# Birdy 文件隐写工具

**完全离线 · 浏览器端高强度文件隐写工具**

将任意文件通过 AES-256-GCM 强加密后，隐写到普通 PNG 或 JPEG 图片中。  
生成的图片外观完全正常，可以正常预览和分享，只有知道正确密码的人才能提取出原始文件。

**在线体验地址**（GitHub Pages 已上线）：  
https://cloudy2791-1.github.io/Birdy-File-Steganography/

---

## ✨ 主要特性

- 完全离线运行，所有运算在浏览器本地完成，无任何网络请求
- 高强度加密：AES-256-GCM + PBKDF2（最高支持 1,000,000 次迭代）
- 高级隐写技术：
  - PNG：zTXt 压缩文本块 + 随机会话 ID + 随机关键词（极隐蔽、无固定特征）
  - JPEG：尾部追加（可选保留原始 JPEG 格式）
- 元数据全部加密：文件名、大小、类型、时间等不留明文
- 自动完整性校验：SHA-256 哈希验证，防止数据被篡改
- 大文件支持：分块加密，理论无大小限制（建议 ≤1GB）
- 安全擦除机制：内存多次覆盖、一键“销毁痕迹”、下载链接 30 秒自动销毁
- 友好界面：20 位随机强密码生成器、实时密码强度检测、进度条显示

## 快速上手（30 秒完成第一次使用）

### 加密并隐写
1. 打开上面的在线地址（或下载仓库中的 `index.html` 双击运行）
2. 切换到「加密并隐写」模式
3. 选择一张**载体图片**（PNG 或 JPEG）
4. 选择你要隐藏的**任意文件**
5. 输入密码（推荐点击「生成20位随机密码」按钮）
6. 选择加密强度（推荐「增强强度 500,000 次」）
7. 点击「开始加密 & 隐写」
8. 下载生成的图片即可完成

### 解密提取
1. 选择之前生成的加密图片
2. 输入正确的密码
3. 点击「解密并验证完整性」
4. 下载恢复的原始文件

## ⚠️ 重要提醒

- 大文件（>100MB）处理时浏览器可能会短暂卡顿，请耐心等待，不要刷新页面
- 生成的图片请**直接使用下载的文件**，如果用图像编辑软件重新保存（尤其是 JPEG），隐写数据可能会丢失
- 忘记密码将无法恢复数据，请妥善保管
- 推荐使用最新版 Chrome / Edge / Firefox 以获得最佳体验
